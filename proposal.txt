--------------------------------------------------------------------------------
RESUMEN EJECUTIVO
--------------------------------------------------------------------------------

Sistema de estacionamiento 100% automatizado con tecnología QR, integración
MercadoPago y dispositivos IoT (ESP32). Operación sin personal para entrada,
pago y salida de vehículos.

--------------------------------------------------------------------------------
ALCANCE DEL PROYECTO
--------------------------------------------------------------------------------

MÓDULO 1: SISTEMA DE ENTRADA
- Generación automática de código QR al ingresar
- Reconocimiento de patentes con cámara LPR
- Asignación automática de espacio disponible
- Barrera controlada por ESP32
- Pantalla informativa con instrucciones

MÓDULO 2: SISTEMA DE SALIDA
- Escaneo de QR en totem de salida
- Cálculo automático de tarifa por tiempo
- Integración con MercadoPago (QR, tarjeta, efectivo)
- Apertura automática de barrera post-pago
- Generación de comprobante digital

MÓDULO 3: PANEL DE CONTROL WEB
- Dashboard en tiempo real
- Mapa visual de espacios por piso/zona
- Monitoreo de dispositivos IoT
- Gestión de tarifas y horarios
- Reportes y analytics
- Alertas y notificaciones

MÓDULO 4: HARDWARE IoT (ESP32)
- Controladores de barreras entrada/salida
- Lectores QR integrados
- Pantallas informativas
- Sensores de ocupación por espacio
- Cámaras LPR para reconocimiento de patentes

MÓDULO 5: INTEGRACIONES
- MercadoPago (pagos QR, tarjetas, punto de venta)
- Sistema de facturación electrónica AFIP
- Notificaciones por email/SMS
- API REST para integraciones futuras

--------------------------------------------------------------------------------
STACK TECNOLÓGICO
--------------------------------------------------------------------------------

BACKEND:
- Node.js + Express / NestJS
- PostgreSQL (base de datos)
- Redis (cache y colas)
- MQTT (comunicación IoT)

FRONTEND:
- Next.js 16 + TypeScript
- Tailwind CSS
- Diseño responsive

MOBILE (opcional):
- React Native / Flutter
- App para usuarios frecuentes

HARDWARE:
- ESP32 para controladores
- Raspberry Pi para procesamiento local
- Cámaras IP con LPR

INFRAESTRUCTURA:
- AWS / DigitalOcean
- Docker + Docker Compose
- CI/CD automatizado
